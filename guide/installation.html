# 安装

Node 版本要求

Vue CLI 需要 Node.js 8.9 或更高版本 (推荐 8.11.0+)。你可以使用 nvm 或 nvm-windows 在同一台电脑中管理多个 Node 版本。

npm install vue-designer --save
# OR
yarn add vue-designer --save

创建一个项目：

import Vue from 'vue';
import VueDesigner from 'vue-designer';
import App from './App.vue';

Vue.use(VueDesigner);

new Vue({
  el: '#app',
  render: h => h(App)
});
