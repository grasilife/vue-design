指南

了解设计指南，帮助产品设计人员搭建逻辑清晰、结构合理且高效易用的产品

易于扩展

它的插件系统可以根据常见需求构建和共享可复用的解决方案。

组件

使用组件 Demo 快速体验交互细节；使用前端框架封装的代码帮助工程师快速开发。

主题

可视化定制和管理站点主题、组件样式

面向未来

为现代浏览器轻松产出原生的 ES2015 代码，或将你的 Vue 组件构建为原生的 Web Components 组件。

# 起步

安装：

npm install vue-designer --save
# OR
yarn add vue-designer --save

创建一个项目：

import Vue from 'vue';
import VueDesigner from 'vue-designer';
import App from './App.vue';

Vue.use(VueDesigner);

new Vue({
  el: '#app',
  render: h => h(App)
});
